================================================================================
                                                              *torchinfo.nvim*

This plugin takes a python source code file and outputs information about each
torch.nn.Module class defined in it, such as number of parameters and FLOPs.

                                                          *torchinfo.get_info*
torchinfo.get_info({opts})

    Import a python source file and outputs info about each torch.nn.Module
    defined inside. This is done asynchronously. When the output is ready,
    a new window is created at the bottom of the screen with the results. >

        torchinfo.get_info({
            file_path="path/to/python/file.py",
            gpu=0
        })
<
    Parameters: ~
        {opts} (table)  Input arguments:
                        - "file_path" (string) Path to a python file.
                        - "gpu" (int) GPU id to compute FLOPs. Use a negative
                          number to use the CPU. Default value is -1 (CPU).

vim:tw=78:ts=8:ft=help:norl:
